## CUSTOM 404-page – Monster Theme 🧟

A fun custom 404 error page featuring a red monster illustration holding a signboard.  
This page is displayed when a user navigates to a non‑existent route.

-------------------

## 🚀 Features ✨

- Dark themed background with centered layout
- Monster illustration (`monster-red.png`)
- Clear error message: *"Oops! Page not found 👀"*
- Playful tagline: *"404 – The monster ate this page. Let's head back!"*
- Responsive design for mobile screens
- Styled "Back to Home" button with hover effect

-----------------------------------

## 🛠️ Technologies 🛠

- HTML5
- CSS3 (Flexbox, media queries)
